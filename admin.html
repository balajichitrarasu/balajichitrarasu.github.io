🐙 Connect to GitHub

To make changes go live automatically, connect your GitHub account. This is a one-time setup — takes 2 minutes.

  • 1Go to github.com/settings/tokens/new
  • 2Give it a name like "Portfolio Admin", set expiry to "No expiration"
  • 3Under Scopes, tick only repo (full control of repos)
  • 4Click "Generate token" and copy the token that appears
  • 5Paste it below and click Connect
⚠️ Important: The token is stored only in your browser. Never share it. It gives write access to your repo.
